# ReasonStack UI

A reliable, professional frontend for inspecting AI agent reasoning coherence. 

## Features

- **Dark Mode Only**: Designed for serious, prolonged research usage.
- **Trust Evaluation**: Visually demonstrates "Fragile" vs "Robust" consensus based on reasoning diversity.
- **Interactive Inspection**: Expand/Collapse detailed reasoning paths and assumptions.
- **Demo Mode**: built-in scenarios to demonstrate the core value proposition.

## Tech Stack

- React (Vite)
- Tailwind CSS
- Framer Motion

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
